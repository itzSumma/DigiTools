# DigiTools

## Project Name
DigiTools

## Little Description
DigiTools is a responsive digital tools marketplace landing page project built with React. In this project, I tried to create a clean and modern frontend based on a Figma-style design where users can explore digital products, add them to cart, remove them, and see checkout notifications.

## Project Purpose
The main purpose of this project is to practice frontend development with React and improve my skills in component design, responsive layout, state management, and UI implementation from a design reference.

This project can be useful for users because it shows digital tools and resources in a simple and organized way. It can also help beginner developers understand how a product section, cart system, toast notification, and responsive landing page can be built in a real project.

## Technologies Used
- React
- Vite
- Tailwind CSS
- DaisyUI
- React Icons
- React Toastify
- JSON Data

## 3 Features About My Project
1. Product and Cart Toggle
Users can switch between the Products and Cart sections from the main section. By default, the products section is shown.

2. Cart Functionality
Users can add products to cart, remove selected products, see the total price, and clear all products by clicking the Proceed to Checkout button.

3. Responsive Design
The project is designed to work on mobile, tablet, laptop, and desktop devices with a modern Figma-inspired UI.

## What I Did In This Repository
In this repository, I created a complete landing page section by section. I made separate React components for the navbar, banner, stats section, product section, cart section, steps section, pricing section, call-to-action section, and footer.

I also used JSON data to show products dynamically, used images from the assets folder, added hover effects and animated card borders, and implemented toast notifications for add to cart, remove, and checkout actions.

## How This Project Can Help Users
This project can help users explore premium digital tools in an easy and attractive interface. It gives a simple idea of how a digital marketplace homepage can look and how users can interact with products before checkout.

For students and beginners, this project is helpful as a practice example of:
- React component structure
- props and state handling
- rendering JSON data
- cart interaction logic
- responsive design
- UI styling and animation
